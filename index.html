Projekty na GitHub

Aplikacja księgowa

Aplikacja księgowa, która w sposób automatyczny liczy podatki z wprowadzonych do serwisu faktur.

Gra w wisielca

Klasyczna gra w wisielca polegająca na odgadywaniu słów z kategorii przysłowia polskie.

Gra w pamięć

Klasyczna gra w pamięć z wykorzystaniem postaci komputerowej kolekcjonerskiej gry karcianej.

Driver

Aplikacja "backendowa" wykorzystująca interfejs REST API dla serwisu samochodowego zajmującego się szkoleniem kierowców.

Oddaj w dobre ręce

Aplikacja wykorzystująca wzorzec architektoniczny MVC. Służy do rejestracji niepotrzebnych rzeczy i przekazania ich zaufanym instytucjom.

Zaplanuj jedzonko

Aplikacja webowa wykorzystująca wzorzec architektoniczny MVC służąca do planowania posiłków w ciągu dnia.

Lista zadań

Klasyczna lista zadań, która jest zapisywana automatycznie w przeglądarce po stronie użytkownika w Local Storage. Aplikacja została napisana w JavaScript z wykorzystanie biblioteki React.js.

Kalkulator walut

Klasyczny kalkulator walut, który przelicza kwotę podaną w złotówkach na dolara amerykańskiego, euro lub funta brytyjskiego. Aplikacja została napisana z wykorzystaniem JavaScript oraz biblioteki React.js.
